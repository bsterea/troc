# Contributing to TROC

TROC is an open project.

We welcome:

- developers
- designers
- translators
- testers
- farmers
- electricians
- teachers
- mechanics
- community builders

You do not need permission to suggest ideas.

You do not need permission to report problems.

Good ideas matter more than titles.

## Development Principles

- direct cooperation
- voluntary participation
- transparency
- respect
- practical solutions

## How To Help

- report bugs
- improve code
- improve translations
- improve documentation
- test new features
- propose new exchange models

Every useful contribution is welcome.
