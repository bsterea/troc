# Troc

Troc este o aplicatie civica pentru schimburi directe intre persoane fizice din Romania.

Aplicatia nu permite vanzarea cu bani, nu permite accesul persoanelor juridice si nu functioneaza ca moneda, criptomoneda sau platforma comerciala.

Scopul proiectului este sprijinirea economiei locale, a gospodariilor, a producatorilor mici si a schimbului direct intre oameni.
# Troc

Troc is a civic application designed to support direct exchanges between private individuals in Romania.

The platform allows people to exchange goods and services without using money, bank cards, cryptocurrencies, or commercial intermediaries.

Troc is not a marketplace, not a payment system, and not a digital currency.

Key principles:

- Private individuals only
- No companies or legal entities
- No money transactions
- No commissions
- No advertising-driven business model
- Support for local communities and local production
- Voluntary and direct exchanges between users

The project aims to strengthen local resilience, encourage self-sufficiency, and create a digital infrastructure for community-based exchange.

Status: Concept and planning phase.
