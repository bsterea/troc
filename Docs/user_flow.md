# User Flow

## Registration

1. User creates an account.
2. User accepts Terms and Privacy Policy.
3. User confirms email address.
4. User completes profile.

---

## Profile Setup

User provides:

- First name
- Last name
- County
- City

---

## Publish Item

1. User creates a new item.
2. User uploads photos.
3. User selects category.
4. User enters title and description.
5. User assigns an estimated Troc value.
6. Item becomes visible.

---

## Search

1. User searches for items.
2. User filters by location and category.
3. User opens item details.

---

## Exchange Offer

1. User selects an item.
2. User proposes an exchange.
3. Owner receives notification.
4. Owner accepts or rejects.

---

## Messaging

1. Users exchange messages.
2. Users agree on exchange details.
3. Users arrange meeting location.

---

## Completion

1. Exchange takes place.
2. Both users mark exchange as completed.
3. Item status becomes exchanged.

---

## Account Deletion

1. User requests account deletion.
2. System removes personal data.
3. Account is permanently deleted.
