# MVP Scope

## Goal

The first version of Troc must allow a private individual to create an account, publish an item, search for items, and propose a direct exchange with another private individual.

---

## Included in MVP

### Account

- Register
- Login
- Logout
- Edit profile
- Delete account

---

### Items

- Create item
- Edit item
- Delete item
- Upload photos
- Mark item as available
- Mark item as exchanged

---

### Search

- Search by keyword
- Filter by county
- Filter by city
- Filter by category

---

### Exchange

- Send exchange offer
- Accept exchange offer
- Reject exchange offer
- Cancel exchange offer

---

### Messages

- Send message
- Receive message
- View conversation

---

## Not Included in MVP

The following features are intentionally excluded from the first version:

- Payments
- Troc wallet
- Ratings
- Reviews
- Advertising
- Companies
- Legal entities
- AI recommendations
- Delivery system
- Complex moderation
- Automatic tax calculation
- Public marketplace for commercial sellers

---

## MVP Success Criteria

The MVP is successful if two private individuals can:

1. Create accounts
2. Publish items
3. Find each other
4. Communicate
5. Agree on a direct exchange
6. Mark the exchange as completed
